# Template de Solicitacao

Use este modelo para registrar novas demandas do TacCheck.

## Titulo

Descrever em uma frase o que deve mudar.

## Objetivo

Explicar o motivo operacional da solicitacao.

## Regra nova ou alterada

Descrever a regra de forma objetiva.

Exemplo:

```text
A velocidade indicada deve ser obtida pelo topo do registro da velocidade.
```

## Fluxo esperado

1. Passo operacional 1.
2. Passo operacional 2.
3. Passo operacional 3.

## O que remover ou evitar

Listar comportamentos antigos que nao devem continuar.

## Testes obrigatorios

Listar os casos que devem continuar passando ou ser criados.

## Evidencias esperadas

Listar prints, imagens marcadas ou links que devem comprovar a entrega.

## Publicacao

Informar se deve:

- apenas alterar localmente;
- commitar;
- enviar para GitHub;
- atualizar GitHub Pages;
- criar tag.
